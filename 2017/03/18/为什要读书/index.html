为什么要读书？

发表于 2017-03-18

「为什么要读书？」这个问题绝对不是不读书人的借口，而是那些经常读书的人的自我反思。观察一下周围，大量的新生事物争抢着我们有限的精力和时间。而读书的必要性以及重要性越来越显得不足挂齿，一个人不读书也可以过得很好，读了书也未必就会更好。读书究竟是为了什么？

获取知识：读书获取知识几乎是每个人的常识，文字和语言使得人类文明得以传承，而书是文字的载体，重要性不言自明。只是随着社会的发展，代替书籍的载体越来越多。一个视频文件在提供图像的同时还提供了声音，一个音频文件可以在不影响做其他事情的同时提供各种信息。这样对比下来，读书获取知识这种途径在效率上有着先天的不足。

消磨时间：不排除一些热爱读书的人在书本中获得的快感会大于在影视作品中获得的快感。只是这个比例非常小，而且并非臆测，看看畅销书的销量和电影的票房就知道了。而这些靠读书消磨时间的人中，还有一部分认为自己是在学习。在选择如此丰富的今天，读书在消磨时间上并不是一个好方式。当然有些书从出现的那一刻就是为了消磨时间的，只是我现在说的并不是那类书。

获取信息：术并不是一个时效性很高的媒介，大多数记录的是一些过往的事件或者经验。而信息讲求时效性，故而书能提供的帮助有限，万维网更适合一些。

上面这几个能想到的原因都被我否定了，我也不知道为什么要读书。只是这个否定并没有立刻产生效果，在无数个不经意间，我还在努力寻找答案。有一天，我看到一句话：「书是一个人系统思想的完整体现」。很普通的一句话，但是那一刻它帮我拨开云雾，找到了答案。读书就是读一个人的思想，读一个人的灵魂，完成一次跨越空间和时间的对接，以此站在巨人的肩上。
